# gtk-raylib
Es una pequeña biblioteca para crear una incrustación de raylib en gtk+3.x.

- Gtk+-3.x
- LibEpoxy : https://github.com/anholt/libepoxy
- OpenGL 3.3
- Raylib 3.x : https://github.com/raysan5/raylib

# Compilar gtk-raylib

- cd src
- sh build-lib.sh

# Compilar ejemplos

- cd examples
- make
